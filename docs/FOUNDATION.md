# Xsaas Foundations

## What we are selling

We are not selling "a bot that spams X".

We are selling:

- a growth copilot for X
- a system that finds what matters faster
- a cleaner workflow for posting, quoting, and replying
- a membership product with dashboard and workspace control

## Best first customer

Start with one segment only:

- creators

Why:

- faster sales cycle
- easier onboarding
- less support than agencies
- clearer value proposition

## Core promise

Xsaas helps you publish smarter on X by surfacing good opportunities, creating sharper drafts, and keeping your workflow organized in one place.

## MVP rules

- draft-first, not full autopilot
- one user can own one or more workspaces
- each workspace can later connect one X account
- membership controls limits and features
- focus on speed, clarity, and publish quality

## First plans

Starter:

- 1 workspace
- idea feed
- draft posts
- basic quote/reply suggestions

Pro:

- more workspaces
- stronger content memory
- scheduling
- better analytics

Agency:

- multiple workspaces
- client handling
- team seats later

## Tech reality with no budget

- MongoDB Atlas free tier is fine for the first stage
- Render is okay to validate, not ideal forever
- PayPal is enough to start taking subscriptions
- do not overbuild before first paying users

## Non-negotiables

- no fake generic images
- no uncontrolled spam loops
- no "fully automatic" promise on day one
- build for trust first
